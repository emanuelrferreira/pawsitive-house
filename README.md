## Overview  
Pawsitive House is a fictional pet care company website designed to demonstrate responsive web design, form validation, and JavaScript interactivity.  
This project was developed as part of the Web Design and Client-Side Scripting module (worth 60% of the final grade) at the National College of Ireland.

## Features  
- Responsive layout using **Bootstrap** and custom CSS  
- Form validation using **HTML5** and **JavaScript**  
- Interactive elements developed with **jQuery**  
- Consistent navigation system across pages  
- SEO and performance optimisation  
- Deployed website version
